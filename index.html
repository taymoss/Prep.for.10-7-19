D3 Elements

There are multiple elements of d3 that need to be used so that your code is made easier. They are usually put together by strings. These strings look like: 
 d3.select("body").append("p").text("Paragraph")
Now I will break down each element of this string.

select("body")

This selects whatever part you specify inside your code. In this case it selects the entire body of the html.

append("p")

This creats a new element and places it at the end of the selection that it is attached with. In this case an empty paragraph was added to the body of the html.

text("New paragraph!")

this element takes a string and places it in the new element created by the string. In this case it places the words new paragraph in the p element.
